# Holidae Website - Web Design Assignment
 Project Overview
This repository contains the source code for a fully responsive website developed as part of the WebDev S2 assignment.

The goal of this project was to build a semantic, accessible, and responsive web interface that strictly adheres to the client requirements checklist. The development follows a "Requirements-First" approach, ensuring a stable product without unnecessary scope creep.

1. Features
Responsive Layout: Adapts seamlessly to Mobile, Tablet, and Desktop screens using CSS Flexbox.
Semantic HTML5: Uses proper tags (`<nav>`, `<main>`, `<section>`, `<footer>`) for better accessibility and SEO.
Interactive Mobile Menu: A lightweight JavaScript implementation for the mobile navigation toggle.
Clean Design: Minimalist UI with a focus on readability and visual hierarchy.

2. Technologies Used
HTML5 (Structure)
CSS3 (Styling & Animation)
JavaScrip (DOM Manipulation)
Google Fonts (Typography)
